## 7-Day Follow-Up Sequence
### Day 0 - Onboarding
- Confirm login and workspace access.
- Connect Amazon account.
- Sync listings.
- Add 1-3 ASINs.
- Run at least one core action (keyword refresh or product refresh).

### Day 2 - Check-In
- Ask if setup remained stable.
- Confirm they could run core tools independently.
- Capture first confusion points.

### Day 5 - Value Check
- Ask which tool gave most practical value.
- Ask where trust is low or results are unclear.
- Capture must-have missing feature.

### Day 7 - Feedback + Payment Question
- Ask for final score.
- Ask if they would continue using weekly.
- Ask if they would pay now and at what level.
- Capture blockers before recommending to others.

## Manual Monitoring Per User
- Amazon connection success.
- Listing sync success.
- Keyword refresh behavior.
- Buy Box check behavior.
- BSR refresh behavior.
- Pincode check outputs (including null/failed handling behavior).
- False alert presence.
- Confusing UI points noted by user.

## Success Criteria (Per User)
- Connects Amazon successfully.
- Adds 1-3 ASINs.
- Runs at least 3 tools.
- Gives feedback.
- Identifies which tool is most valuable.
- States whether they would pay.
